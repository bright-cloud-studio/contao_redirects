# Bright Cloud Studio's Contao Redirects
This was a temporary package to get Asconsulting's Redirect Manager package into Contao 5+. He has since updated his original package, making this redundtant. Updating for those who have this installed so things dont fall apart. For new users, just use [Redirect Manager](https://github.com/asconsulting/redirect_manager).
